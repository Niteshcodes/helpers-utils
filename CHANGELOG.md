# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-03-27

**Author:** Nitesh Upadhayay ([@NiteshCodes](https://github.com/NiteshCodes))

### Added
- Initial release of @nitesh-upadhayay/helpers-utils
- Type Checks module (isBool, isNegative, isEmail, isPrime, etc.)
- String Casing & Manipulation module (camelCase, snake_case, kebab-case, etc.)
- Array Utilities module (chunk, flatten, shuffle, unique, etc.)
- Object Utilities module (merge, pick, omit, deepClone, etc.)
- Math Utilities module (round, random, factorial, fibonacci, etc.)
- Date & Time Utilities module (addDays, format, getAge, etc.)
- Promise & Async Utilities module (retry, delay, debounce, throttle, etc.)
- Validation Utilities module (isStrongPassword, isCreditCard, isUuid, etc.)
- Color & Format Utilities module (hexToRgb, formatBytes, formatCurrency, etc.)
- Encoding & Encryption Utilities module (toBase64, toHex, caesarCipher, etc.)
- Browser/DOM Utilities module (getQueryParam, setClipboard, localStorage, etc.)
- Express/Server Utilities module (sendSuccess, getClientIp, validateRequired, etc.)
- File & Path Utilities module (join, resolve, basename, dirname, etc.)
- Environment Utilities module (getEnv, isDevelopment, loadEnv, etc.)
- Zero dependencies
- Full TypeScript support
- Tree-shakeable
- CommonJS and ESM support
